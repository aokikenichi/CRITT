Cricket Guide / Equipment 01

三本のStumps、
二本のBails

クリケットのウィケット、その形と進化

クリケットでいうwicketは、三本のstumpsと、その上に載る二本のbailsからなる道具一式です。基本の木製wicketから、設置環境、安全性、初心者への配慮、LEDやカメラなどの技術に応じた形まで紹介します。

THREE STUMPS,
TWO BAILS.
Definition

まず、
呼び方から。

一本一本はstump、上に載る一本一本はbail。そして、三本のstumpsと二本のbailsを合わせた一式をwicketと呼びます。

Wicket = 3 Stumps + 2 Bails

このページでは一式としてのwicketを紹介しつつ、個別部品名のstumps、bailsも併記します。

Six Forms

6つの
代表例。

正式試合の基本形から、練習・教育用、LEDやカメラを備えたタイプまでを見ていきます。

STANDARD01

木製の標準wicket

正式試合の基本形木製

三本の木製stumpsを地面に刺し、その上に二本のbailsを載せる基本形です。

  • 高さや全体幅などが規則で定められる
  • 学校、クラブ、リーグ、上位大会で使用
情報ソース：MCC Laws: The Wickets
ENVIRONMENT-READY02

台座付き・設置対応wicket

主に練習・体験会屋内対応

体育館や舗装面など、stumpsを地面に刺せない場所で自立するよう台座を備えています。

情報ソース：FORTRESS Rubber Base Stump
SAFETY / RETURN03

バネ付き・復元型wicket

主に練習用復元型

ボールが当たると傾き、バネなどの機構で元に戻ります。

情報ソース：Gray-Nicolls Spring Return Stumps
BEGINNER / SCHOOL04

プラスチック製・初心者向けwicket

教育・普及向け軽量

軽く、運びやすく、視認性の高い色を使うことも多い導入用のwicketです。

情報ソース：Gray-Nicolls Power Play Plastic Set
TECH / LED05

LED wicket / LED bails

承認条件のもとで正式試合可発光

bailが外れた瞬間に光るタイプです。判定を見やすくし、中継や夜間試合の演出にも使われます。

情報ソース：Zings official site
TECH / CAMERA06

stump camera内蔵wicket

承認条件のもとで正式試合可放送技術

stumpの内部にカメラやマイクを搭載し、wicketに近い位置から迫力ある映像や音を届けます。

情報ソース：Science Museum Group: CCD cricket stump camera

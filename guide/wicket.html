Wicket forms & evolution

三本のStumps、
二本のBailsクリケットのウィケット、その形と進化

クリケットでいう wicket は、三本の stumps と、その上に載る二本の bails からなる道具一式です。

基本の木製wicketから、設置環境、安全性、初心者への配慮、LEDやカメラなどの技術に応じた形まで、代表的な6種類をオリジナルの解説イラストとともに紹介します。

wicketの仕様は競技形式や製品によって異なります。各項目のリンク先もあわせてご確認ください。

Orientation

まずは
見取り図。

ここで紹介する対象は、stumps単体ではなく、stumpsとbailsを合わせたwicket一式です。分類は「公式・非公式」の区分ではなく、wicketが何を解決するために変化したかに注目しています。正式試合で使えるかどうかは、それぞれの項目のラベルとして添えています。

標準型wicket は、木製の三本のstumpsを地面に刺し、二本のbailsを載せる基本形です。そこから、硬い地面や屋内への対応、安全性、初心者への配慮、そしてLEDやカメラなどの機能拡張が生まれてきました。

特に LED型 や カメラ内蔵型 は、wicketの基本構造を保ちながら機能を加えた例です。承認条件のもとで正式試合でも使われることがあります。

標準型木製・三本・bails付きの基本形。
設置対応型刺せない地面や室内でも使える。
安全・復元型練習のしやすさと安全性を優先。
初心者・教育型軽量・安価・扱いやすい。
LED型離脱時に発光し判定と演出を補助。
カメラ内蔵型stumpの視点から映像を届ける。

ここでは、代表的な6種類のwicketを順番に見ていきます。各カードには、stumpsとbailsの構造、主な使われ方、そして元になった情報ソースへのリンクを付けました。

STANDARD TYPE01

木製の
標準wicket

正式試合の基本形木製屋外用

もっとも基本となるstumpsです。3本の木製stumpsに2本のbailsを載せ、芝や土の上に刺して使います。多くの人が思い浮かべる「クリケットのwicket」はこの形です。

  • 三本が独立している
  • bailsを上に載せる
  • 高さや全体幅などに標準規格がある

主な場面：学校、クラブ、リーグ、上位大会など。

  • MCC Laws of Cricket: The Wickets
ENVIRONMENT-READY02

台座付き・
設置対応wicket

主に練習・体験会向け台座付き屋内対応

地面に刺せない場所でも使いやすいように、台座で自立するタイプです。体育館や舗装面、イベント会場など、通常のstumpsが使いにくい環境で活躍します。

  • 刺さなくても立つ
  • 硬い床や室内でも使える
  • 標準型に近い見た目を保つことが多い

主な場面：室内練習、学校体育、体験会、刺せない地面での簡易ゲーム。

情報ソース
  • FORTRESS Rubber Base Cricket Target Stump
SAFETY / RETURN03

バネ付き・
復元型wicket

主に練習向けバネ式復元型

当たると倒れたり傾いたりしつつ、元の位置に戻りやすいタイプです。立て直す手間を減らせるので、反復練習に向いています。

  • 復元機構がある
  • 練習のテンポがよくなる
  • 子どもや初心者にも扱いやすい

主な場面：投球練習、レクリエーション、設営の手間を減らしたい練習環境。

情報ソース
  • Gray-Nicolls: Spring Return Stumps
BEGINNER / SCHOOL04

プラスチック製・
初心者向けwicket

教育・普及向け軽量プラスチック

軽くて扱いやすい、普及・導入向けのstumpsです。カラフルで視認性が高く、学校やジュニア教室で使いやすいのが特徴です。

  • 軽量で持ち運びしやすい
  • 価格を抑えやすい
  • 初心者や子どもでも扱いやすい

主な場面：学校授業、ジュニア教室、体験イベント、普及プログラム。

情報ソース
  • Gray-Nicolls: Power Play Plastic Set
  • MACE: Plastic Stump with Rubber Base
TECH / LED05

LED wicket /
LED bails

承認条件のもとで正式試合可発光中継向け

bailが外れた瞬間に光るタイプです。判定の補助になるだけでなく、夜間や中継試合では視覚的な演出効果も高く、現代的なstumpsの代表例です。

  • bailsが発光して離脱を見やすくする
  • 観客や中継にとって印象的
  • テクノロジー型でも正式試合に入ることがある

主な場面：国際大会、テレビ中継のある試合、演出性の高い試合。

情報ソース
  • Zings official site
TECH / CAMERA06

stump camera
内蔵wicket

承認条件のもとで正式試合可撮影機能放送技術

stumpの内部にカメラやマイクを搭載し、通常の視点では見られない迫力ある映像を実現するタイプです。カメラは中央のstumpなどに収められ、三本の外径をそろえたwicketとして使われます。

  • wicketに近い視点からプレーを撮影できる
  • リプレーやハイライトの臨場感を高める
  • 競技と放送技術の融合を象徴する存在

主な場面：上位大会、中継試合、ハイライト映像の強化。
